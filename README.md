# 🦠 COVID-19 Death Rate Predictor

A machine learning web application that predicts whether a country will have a HIGH (>3%) or LOW (≤3%) COVID-19 death rate based on population, testing data, and case statistics.

## 🚀 Live Demo

**Try it here:** https://covid-death.streamlit.app/

## 📊 Features

- Predicts death rate category (High/Low) with 68% accuracy
- Built with Decision Tree Classifier
- Interactive web interface with Streamlit
- Trained on real COVID-19 data from 173 countries

## 🛠️ Tech Stack

- Python
- scikit-learn (Machine Learning)
- Pandas (Data Processing)
- Streamlit (Web Interface)
- Deployed on Streamlit Cloud

## 📈 Model Performance

- **Accuracy:** 68.57%
- **Algorithm:** Decision Tree Classifier
- **Features Used:** Population, Total Tests, Tests per Million, Cases per Million
